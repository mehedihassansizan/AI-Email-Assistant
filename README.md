# 🤖 AI Email Assistant

An AI-powered email automation workflow built with **n8n**, **Gmail**, and **Groq LLM**. This workflow automatically detects incoming emails, processes them using an AI Agent, and sends intelligent replies through Gmail.

---

## 🚀 Features

- 📩 Automatically detects new Gmail messages
- 🤖 AI Agent analyzes email content
- 💬 Generates context-aware responses using Groq LLM
- 📤 Automatically sends replies via Gmail
- ⚡ No manual intervention required

---

## 🛠️ Tech Stack

- n8n
- Gmail Trigger

- Workflow Automation

## ⚙️ How It Works

1. A new email arrives in Gmail.
2. The Gmail Trigger detects the email.
3. The AI Agent processes the email using the Groq Chat Model.
4. A context-aware reply is generated.
5. Gmail automatically sends the response.
